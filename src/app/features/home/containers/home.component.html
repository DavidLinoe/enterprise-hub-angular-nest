Plataforma de gerenciamento de empresas, usuários e máquinas com autenticação JWT e segurança em nível de linha (RLS) no PostgreSQL.

Empresas

Cadastre e gerencie empresas com CNPJ. A base de tudo no sistema.

Usuários

Vinculados a uma empresa, com papéis ADMIN ou USER.

Máquinas

Equipamentos com número de série, organizados por empresa.

Stack utilizada

  • Back-end: NestJS + TypeORM + PostgreSQL + JWT
  • Front-end: Angular + Tailwind CSS
  • Arquitetura em camadas (containers, facades, stores, apis)

Segurança no banco

A proteção dos dados é feita por Row Level Security no PostgreSQL. Usuários com role USER só enxergam dados da própria empresa, e usuários ADMIN têm acesso total — regras aplicadas diretamente pelo banco, não apenas pela aplicação.
